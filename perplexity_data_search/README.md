# Perplexity Data Search

Прототип интеллектуального поиска по каталогу дата-продуктов:
- декомпозиция аналитического запроса;
- извлечение фильтров;
- гибридный поиск BM25 + vector;
- rerank через cross-encoder;
- синтез итогового ответа с обоснованием.

Запуск и полная инструкция: `INSTRUCTIONS_USER.md`.
